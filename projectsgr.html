Μετεωρολογική ιστοσελίδα με OpenWeather API

Το πρότζεκτ του Ιστότοπου Πρόβλεψης Καιρού, διαθέσιμο στη διεύθυνση https://axlna.github.io/, είναι μια ιστοσελίδα σχεδιασμένη για να παρέχει στους χρήστες αξιόπιστες και λεπτομερείς προβλέψεις καιρού. Το έργο χρησιμοποιεί το OpenWeather API ως την κύρια πηγή δεδομένων για τη συλλογή πραγματικού χρόνου πληροφοριών καιρού, εξασφαλίζοντας ακριβείς και ενημερωμένες προβλέψεις.

Πέρα από το OpenWeather API, το έργο ενσωματώνει επίσης το Pexels API. Το Pexels API ενισχύει την οπτική εμπειρία του ιστότοπου επιλέγοντας αυτόματα και εμφανίζοντας σχετικές εικόνες που σχετίζονται με την αναζήτηση της τοποθεσίας.

Ο ιστότοπος έχει δημοσιευθεί χρησιμοποιώντας το GitHub Pages.
