Weather Forecast Website utilizing OpenWeather API

The Weather Forecast Website project, where you can visit online and on GitHub, is a web application designed to provide users with reliable and detailed weather forecasts. The project leverages the OpenWeather API as its primary data source to gather real-time weather information, ensuring accurate and up-to-date forecasts.

In addition to the OpenWeather API, the project also incorporates the Pexels API. The Pexels API, enhances the visual experience of the website by automatically selecting and displaying relevant images related to the searched location.

The website has been published using GitHub Pages.
